# jsdoc README

gen jsdoc

## Features

选中一个函数的参数, `ctrl+shift+p` 后输入 `gen jsdoc` 并选择 `gen jsdoc` 命令, 即可生成 jsdoc 风格的函数注释

## Requirements

none

## Extension Settings

This extension contributes the following settings:

- `jsdoc.author`: 添加此选项, jsdoc 将自动插入作者
